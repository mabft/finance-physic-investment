> **版本**: v1.0 | **部署日期**: 2026-06-30 | **覆盖标的**: 18个

---

## 一、系统概述

基于金融物理学（Econophysics）的**全自动每日投资分析系统**，在每个交易日自动执行盘前/盘中/盘后三段式分析，输出四维物理金融指标，给出仓位管理和定投调整建议。

### 核心能力

- 🔬 **四维物理指标**: 温度T（波动率²）、熵H（Shannon信息熵）、动量M（收益/风险比）、Hurst指数（趋势持续性）
- 📊 **7种市场状态**: BULL_TREND / BULL_MEANREV / SIDEWAYS / BEAR_MEANREV / BEAR_TREND / CRISIS / TRANSITION
- 🎯 **0-100标的评分**: 基于四维指标的综合筛选评分系统
- 💰 **仓位管理矩阵**: 市场状态→仓位上限+定投金额自动调整
- 🚨 **买卖信号**: 清仓级(T>0.15)/减仓级/警告级三级预警

## 三、四维物理金融指标

| 指标 | 公式 | 阈值 | 含义 |
|------|------|------|------|
| **温度 T** | σ²(annual) | >0.15🔴临界 >0.08🟠偏热 >0.03🟡温和 ≤0.03🟢正常 | 市场波动烈度 |
| **熵 H** | Shannon (bits) | >2.5🔴混沌 1.5-2.5🟡不确定 <1.5🟢有序 | 信息可预测性 |
| **动量 M** | μ/σ (20日) | >0.5📈强升 0~0.5弱升 -0.5~0弱降 <-0.5📉强降 | 方向强度 |
| **Hurst** | R/S斜率 | >0.55趋势 0.45-0.55随机 <0.45均值回归 | 持续性/反转性 |

## 七、每日分析流程

```
09:10 早盘诊断
  ├── 创建飞书投资日报
  ├── 抓取隔夜美股+港股+新闻
  ├── 计算四维指标（所有18标的+4大指数）
  ├── 判定综合市场状态
  ├── 写入"物理金融状态诊断"板块
  └── 发送盘前要点+文档链接

12:30 午盘分析
  ├── 对比盘中 vs 盘前指标变化
  ├── 检测波动率飙升/趋势反转/熵骤增
  ├── 追加"午盘分析"到日报
  └── 发送下午操作建议

15:05 收盘分析
  ├── 全量18标的收盘体检
  ├── 逐个标的：四维解读+评分+止损位+具体操作
  ├── 追加"收盘分析"到日报
  ├── 同步飞书Bitable
  └── 发送次日策略

周一 9:00 定投周报
  ├── 4个定投标的物理状态
  ├── 定投金额调整（正常/加码/减量/暂停）
  └── 止盈止损提醒
```

## 十一、注意事项

1. **OTC基金无物理指标**: 场外基金只有净值，没有日内K线，无法计算温度/熵/动量/Hurst
2. **新浪API编码**: 使用GBK编码，需 `iconv -f GBK -t UTF-8` 转换
3. **收盘后数据**: 手动触发需在收盘后（15:00+）才有完整日数据
4. **Hurst需要长序列**: 至少100个数据点，不足时会跳过该指标
5. **温度极高可能失真**: 剧烈上涨导致T异常高（如515980的T=0.62），需结合熵和动量判断
